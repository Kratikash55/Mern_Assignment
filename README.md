# Product Management App

A MERN stack application to manage products with add, edit, delete, publish toggle, and image upload features.

---

##  Overview
This app lets you:
- Add products with details (name, type, stock, MRP, price, brand, description, return eligibility).
- Upload product images.
- Edit, delete, and toggle publish status.
- View products in a responsive grid.
